# How to install
**1.**
`Install qt5 or qt6`

**arch:**
```
sudo pacman -Syu
sudo pacman -S qt6-tools
```
**Debian:**
```
sudo apt update
sudo apt upgrade
sudo apt install qt6-tools
```

**2.**
```
cd My-first-GUI-project
mkdir build
cd build
cmake ..      
make         
./first
```

**That's all!**
